Редактор плану

Камери, двері, вікна, обладнання і текст переносяться у 2 кліки: вибір об'єкта → нова позиція. Стіни редагуються через точки на кінцях.

Стіна: “Додати стіну” → 2 кліки по схемі.
Перемістити стіну: клік по синій точці на кінці стіни → клік у нове місце.
Камера / двері / вікно / обладнання / текст: 1 клік по об'єкту → 2 клік у нову точку.
Видалення: “Видалити” → клік по об'єкту або по точці стіни.
Скинути шаблон: повертає стартову схему.
Очистити все: робить повністю пусте полотно.
